## 📌 About
AI Research Agent is an autonomous AI agent that automatically searches the web and generates comprehensive research reports on any topic — without any manual intervention!

## ✨ Features
- 🤖 Fully autonomous web research agent
- 🌐 Real-time Google search using SerpAPI
- 🧠 Intelligent analysis using Google Gemini AI
- 📥 Download research reports as text files
- ⚡ Quick topic buttons for instant research
- 🎨 Beautiful dark professional UI

## 🔄 How the Agent Works
1. SEARCH — Agent searches Google automatically
2. RETRIEVE — Collects top 5 search results
3. ANALYZE — Gemini AI analyzes all results
4. GENERATE — Creates comprehensive report
5. DELIVER — Shows formatted research to user
